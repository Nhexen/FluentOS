# FluentOS Roadmap

## Version Overview

| Version | Objective | Key Content | Editions |
|---------|-----------|------------|----------|
| **v0.1 POC** | Establish structure and vision | Vision, scope, non-goals, basic installation scripts, theme skeletons | Core (POC) |
| **v0.2 Consistency** | Build core visual/UX layer | Theme refinement, tweaks, basic Seelen/Windhawk integration, profile JSON expansion | Core |
| **v0.3 Business Ready** | Add organizational features | Group Policy templates, company branding, role profiles, service policies, audit logging | Business (POC) |
| **v0.4 Business Branding** | Enhance customization | Advanced branding options, multiple role presets, business-tier automation | Business |
| **v0.5 Fleet Preview** | Introduce fleet features | Multi-machine management, centralized config, Intune integration (preview) | Fleet (POC) |
| **v1.0 Core Platform** | Production release | Stable Core edition, full Business support, Fleet preview, comprehensive docs, public beta | Core, Business, Fleet (Beta) |

- [ ] Vision and scope documentation
- [ ] Installation script framework
- [ ] Theme profile templates (light/dark)
- [ ] Tool integration guides (Seelen, Windhawk, DWMBlurGlass, Explorer)

### v0.2 – Consistency Layer
- [ ] Refined visual design system
- [ ] Complete theme assets
- [ ] Tested Seelen configurations
- [ ] Tested Windhawk mod list
- [ ] DWMBlurGlass effect library

### v0.3 – Business Ready
- [ ] Group Policy templates (.admx)
- [ ] Company branding module
- [ ] Role profile definitions (Standard, Power User)
- [ ] Service optimization policies
