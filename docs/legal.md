## Legal Scope

### What FluentOS Does

FluentOS operates entirely within the **supported Windows APIs and features**:
- Registry modifications (HKEY_CURRENT_USER, HKEY_LOCAL_MACHINE via proper permissions)
- Group Policy objects (gpupdate, gpedit, registry-based policies)
- Windows Settings customization
- Third-party tool configuration and integration
- PowerShell automation and scripting

### What FluentOS Does NOT Do

- Modify or patch the Windows kernel
- Load unsigned drivers
- Circumvent UAC or disable security features without consent
- Redistribute or modify Windows system binaries (*.dll, *.exe from system folders)
- Bypass Windows Defender or other security mechanisms
- Provide access to restricted system functionality
